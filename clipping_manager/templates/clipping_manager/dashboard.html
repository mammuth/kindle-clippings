Random Quote on Page

Your Random Quote URL https://kindle-clippings.us.aldryn.io/en/clippings/random-fullscreen?refresh=60

 

1. As webpage with random quote (auto refreshing)

If you have a screen / TV / wallmount / ... where you want to show random quotes of your kindle highlights you can follow these steps:

  1. Import your clippings here
  2. Modify ?refresh=X in the above URL, where X is your wished refresh interval in seconds
  3. The page will show a new quote every X seconds

2. As New Tab Page

If you want to see a random quote/highlight whenever you open a new tab, follow these steps:

  1. Import your clippings here
  2. Install a browser extension like for example New Tab URL
  3. Add the above Random Quote URL to the settings of the extension
